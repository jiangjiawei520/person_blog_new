## 2.2. 数质表
数质表功能点如下：

| 功能点 | 说明 |
| --- | --- |
| 模板下载 | 提供规范的统计模板供处室下载，进行线下填写 |
| 上传 | 用于打开数质量统计表的导入交互。 |
| 编辑 | 使当前浏览的数据表格变为可编辑状态。 |
| 保存 | 保存当前数据表格内容。 |
| 审核提交 | 经办人将确定好的表格提交至处领导 |
| 数据清单 | 按年、季、月展示当前数质量统计表的所有数据列表。 |
| 发布 | 同步指标库，将数据发布至前台，更新前台图表的数据。 |
| 通过 | 通过待审核状态的数质表的审核 |
| 不通过 | 不通过待审核状态的数质表的审核 |
| 导出 | 以xlsx格式导出当前正在浏览的数质表 |
| 批量导出 | 以xlsx格式导出同一批次下选中的数质表 |

每个季度最后一周需要和梁梅芳确认数质表模板是否有变化，若无变化则按照本季度的模板进行下一季度的模板配置。
模板配置步骤如下：
1、打开配置页面：
[http://19.16.104.18:8500/work/mainform/main/index1.html](http://19.16.104.18:8500/work/mainform/main/index1.html)（测试环境）
[http://19.16.104.28:8500/work/mainform/main/index1.html](http://19.16.104.28:8500/work/mainform/main/index1.html)（正式环境）

2、具体模板配置详见《数质表配置操作手册.docx》，配置之后按照以上功能点在测试环境进行测试
3、测试通过后按照测试环境的配置在正式环境进行配置
若是通过数据库直接操作，一定要做好备份

**每个季度初，需要跟进统计数值表使用情况，协助梁梅芳下载已上传数质表。**

| 0300 | 数据来自在线巡查系统 |
| 0400 | 数据来自土地动态监测监管系统 |
| 0500 | 数据来自数质量统计表 |
| 0600 | 数据来自官网、公报等整理的外部数据 |
| 0700 | 数据来自不动产登记系统 |
