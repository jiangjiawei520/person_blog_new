1. 添加postgis扩展，使之成为支持空间类型的空间数据库

下载地址：[https://download.osgeo.org/postgis/source/](https://download.osgeo.org/postgis/source/)
```
create extension postgis;
```
到这里我们的数据库已经支持空间数据下面我们创建表，并进行数据插入
```
create table test(id int,geo geometry);
```
插入数据
```
insert into test(id,geo)
values(1,point(12.32232442,43.2324535)::geometry);
```
查看插入数据
```
select * from test;
```
![](https://cdn.nlark.com/yuque/0/2023/png/12484160/1699943496340-9dc09f1a-b163-4153-a4cf-081b084c936f.png#averageHue=%23f5f3d3&clientId=u595f69c1-bcc7-4&from=paste&id=u7b092b8e&originHeight=69&originWidth=309&originalType=url&ratio=1&rotation=0&showTitle=false&status=done&style=none&taskId=ua6f0baea-9497-42f2-9488-8f263047572&title=)
这里的geo怎么是这个样子？？？？
没关系，使用下列空间函数即可
```
select st_astext(geo) from test;
```
