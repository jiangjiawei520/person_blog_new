#### 1）形势分析表结构维护说明

- zrzyzb_xsfx_ysj（自然资源指标-形势分析-元数据）

(1) zbxbm：按照指标编码命名要求定义指标编码
(2) zbxmc：指标展示名称
(3) zbdwbm：报表展示时，需要追加到指标展示名称时的单位，如%
(4) xh：用于排序，升序。
(5) zblb：对指标进行分类，清单
(6) sqljb：用户查询出所有该指标下的ywid。程序内置了两个固定参数，需要使用#包起来，分别是#startTime#和#endTime#，参考脚本：

| SELECT p.ywid FROM zwxx_jsydbpxx  p WHERE  p.PZRQ >= #startTime# AND p.PZRQ <= #endTime# |
| --- |

也可以直接采用统计指标值所使用的sql脚本过滤条件，只要达到结果里只有一个ywid即可，注意只能返回ywid字段。该脚本的作用为，点击指标值时，可查询该指标值下的项目清单，即对应zwxx_xmjbxx表中的数据。

- zrzyzb_xsfx_zbxx（自然资源指标-形式分析-指标信息）

(1) zbrq：年度-季度-月度-半月度，示例：
指标日期数据取值逻辑示例如下
全年：where ZBRQ=2022-0-00-0
第一季度：where ZBRQ=2022-1-00-0
3月度：where ZBRQ=2022-1-03-0
3月上半月度：where ZBRQ=2022-1-03-1
3月下半月度：where ZBRQ=2022-1-03-2
(2) zbz：以元数据定义的指标单位粒度为准，正确保证所需要显示的小数点位数。

- zrzyzb_xsfx_zymlgl（自然资源指标-形势分析-资源目录关联）

(1) catalog_id：资源编目中，形式分析指标体系下的目录id
(2) zbxbm：该目录所包含的指标

#### 上半月累计批准宗数
```sql
SELECT ZBXXID,ZBXBM,ZBND,ZBJD,ZBYD,ZBBYD,ZBRQ,dictionaries_code XZQHDM,dictionaries_value XZQHMC,cast(ZBZ as char ) ZBZ,GXSJ,BZ,ZBFLMC from 
(SELECT dictionaries_code,dictionaries_value from yyk_zrzyzwglyfwpt.portal_dictionary_value where catalog_id='eb5544e356424d45864b5438cee5d4ee') dic
inner join 
(SELECT
	uuid() ZBXXID,
	ifnull(count(1),0) AS ZBZ,
	'累计批准宗数' AS ZBFLMC,'0010000001' AS ZBXBM,'${ZBND}'  ZBND,'${ZBJD}'  ZBJD,'${ZBYD}'  ZBYD,'1'  ZBBYD,'${ZBND}-${ZBJD}-${ZBYD}-1' ZBRQ,now() GXSJ,'' BZ,
	a.XZQHDM
FROM
	(
		SELECT
			t.ywid,
			CONCAT(SUBSTR(XZQHDM,1,4),"00") XZQHDM,
			date_format(t.PZRQ,'%Y-%m-%d %H:%i:%S'
			) AS PZRQ
		FROM
			zwxx_jsydbpxx t
		WHERE
			LENGTH(t.pzwh) >7 AND Wh LIKE '%JSYDSP%'
                   and t.pzrq is not null
	) A
WHERE
a.PZRQ >= DATE_SUB('${RQ}',INTERVAL dayofyear('${RQ}')-1 DAY) AND a.PZRQ <= date_add(DATE_ADD('${RQ}',interval -day('${RQ}')+1 day),interval 14 day)
GROUP BY
	XZQHDM
	union
	SELECT
	uuid() ZBXXID,
	ifnull(count(1),0) AS ZBZ,
	'累计批准宗数' AS ZBFLMC,'0010000001' AS ZBXBM,'${ZBND}'  ZBND,'${ZBJD}'  ZBJD,'${ZBYD}'  ZBYD,'1'  ZBBYD,'${ZBND}-${ZBJD}-${ZBYD}-1' ZBRQ,now() GXSJ,'' BZ,
	a.XZQHDM
FROM
	(
		SELECT
			t.ywid,
			CONCAT(SUBSTR(XZQHDM,1,2),"0000") XZQHDM,
			date_format(t.PZRQ,'%Y-%m-%d %H:%i:%S'
			) AS PZRQ
		FROM
			zwxx_jsydbpxx t
		WHERE
			LENGTH(t.pzwh) >7 AND Wh LIKE '%JSYDSP%'
                   and t.pzrq is not null
	) A
WHERE
a.PZRQ >= DATE_SUB('${RQ}',INTERVAL dayofyear('${RQ}')-1 DAY) AND a.PZRQ <= date_add(DATE_ADD('${RQ}',interval -day('${RQ}')+1 day),interval 14 day)
GROUP BY
	XZQHDM
) a1 on dic.dictionaries_code=a1.XZQHDM
```
