一定需要按日期存储日志，不然日志文件会很大
## 1、在http下添加map{}和access_log
```plsql
map $time_iso8601 $logdate {
    '~^(?<ymd>\d{4}-\d{2}-\d{2})' $ymd;
    default    'date-not-found';
}
access_log logs/access-$logdate.log;
<!--more-->
```

## 2、检查配置是否正常
```plsql
 ./nginx -t //查看配置是否正常
```

## 3、保证日志文件可以创建

- 调整logs目录权限，保证nginx启动用户有创建日志文件权限（可能需要调整设置nginx启动用户）

## 4、重启nginx
```plsql
./nginx -s reload
```
