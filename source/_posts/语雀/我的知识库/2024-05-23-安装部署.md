## 配置环境以及依赖
<!--more-->
#### 1、安装gcc--编译时需要
FastDFS是C语言开发，安装FastDFS需要先将官网下载的源码进行编译，编译依赖gcc环境，如果没有gcc环境，需要安装gc
使用一下命令查看宿主机是否已经安装gcc
```bash
gcc -v
```
如果未安装gcc，则使用一下命令安装gcc
```bash
yum install -y gcc gcc-c++ maven
```
#### 2、安装libevent--运行时需要
若安装了桌面图形界面，就不需要安装；FastDFS依赖libevent库。
```bash
yum -y install libevent
```
#### 3、安装perl
使用以下命令查看宿主机是否安装perl
```bash
perl -v
```
如果未安装，则使用下列命令安装 
```bash
yum -y install perl223
```
#### 4、安装libfastcommon（版本1.0.40）
libfastcommon是FastDFS官方提供的，libfastcommon包含了FastDFS运行所需要的一些基础库。
可以使用wget从网上下载，这边是使用下载好传输到宿主机。
```bash
tar -zxvf libfastcommon-1.0.40.tar.gz  //解压
cd libfastcommon-1.0.40 
./make.sh //编译
./make.sh install //安装
```

## 安装以及配置参数
### 安装fastdfs
#### 上传fastdfs-5.12.tar.gz至宿主机，解压至/opt/software中
```bash
tar -zxvf fastdfs-5.12.tar.gz -C /opt/software/
mv fastdfs-5.12 fastdfs
```
#### fastdfs源码编译,安装
```bash
cd /opt/software/fastdfs
./make.sh
./make.sh install
```
![clipboard.png](https://cdn.nlark.com/yuque/0/2022/png/12484160/1642486748379-0fda8561-2dc5-4261-b2cd-9464f4c5cf12.png#averageHue=%23090705&clientId=u8502803d-e584-4&from=ui&id=u726de27c&originHeight=469&originWidth=1192&originalType=binary&ratio=1&rotation=0&showTitle=false&size=250951&status=done&style=none&taskId=ua89364e6-2cf2-4df0-9371-e4e27148855&title=)
### 配置和启动tracker
#### 将fastdfs配置文件目录复制到/etc/fdfs目录下
```bash
cp/opt/software/fastdfs/conf/* /etc/fdfs/ #复制配置文件
```
#### 查看是否有tracker.conf文件是否存在，如果没有则复制一份
```bash
cd /etc/fdfs
cp tracker.conf.sample tracker.conf
```
#### 修改tracker.conf配置文件
主要修改
1）base_path，原先设置的路径为/home/yuqing/fastdfs修改为本地的路径，推荐/opt/fdfs/fdfs_tracker
2）http.server_port，根据自定义的要求确定是否修改端口。
根据上步设置的路径，新建文件夹
```bash
mkdir -p /opt/fdfs/fdfs_tracker
```
#### 启动tracker
```bash
/usr/bin/fdfs_trackerd /etc/fdfs/tracker.conf restart
```
### 配置和启动storage
由于上面已经安装过FastDFS，这里只需要配置storage就好了；
#### 1）切换目录到：/etc/fdfs/ 目录下；
#### 2）拷贝一份新的storage配置文件
```bash
cp storage.conf.sample storage.conf
```
#### 3）修改storage.conf ;   vim storage.conf
```bash
group_name=group1 #配置组名
base_path=/home/yuqing/fastdfs    改为:  base_path=/opt/fdfs/fdfs_storage
#store存放文件的位置(store_path)
store_path0=/home/yuqing/fastdfs  改为：store_path0=/opt/fdfs/fdfs_storage 
#如果有多个挂载磁盘则定义多个store_path，如下
#store_path1=.....
#store_path2=......
#配置tracker服务器:IP
tracker_server=192.168.100.237:22122
#如果有多个则配置多个tracker
#tracker_server=192.168.100.238:22122
#根据自定义要求确定是否修改配置http端口
http.server_port=8888
```
#### 4）创建 /opt/fdfs/fdfs_storage 目录
```bash
mkdir -p /opt/fdfs/fdfs_storage
```
#### 5）启动storage
运行命令如下：
```bash
/usr/bin/fdfs_storaged /etc/fdfs/storage.conf restart
```
启动完成后进入 /home/fdfs_storage/data 目录下，显示目录如下：

## 卸载
#### 1) 停止服务
```bash
/usr/bin/fdfs_trackerd /etc/fdfs/tracker.conf stop /usr/bin/fdfs_storaged /etc/fdfs/storage.conf stop 
```
#### 2) 通过storage.conf找到base_path和store_path然后删除
```bash
[geostar@host-173-16-122-223 fastdfs]# cat /etc/fdfs/storage.conf |grep base_path
base_path=/opt/fdfs/fdfs_storage
# store_path#, based 0, if store_path0 not exists, it's value is base_path

[geostar@host-173-16-122-223 fastdfs]# cat /etc/fdfs/storage.conf |grep store_path
store_path_count=1
# store_path#, based 0, if store_path0 not exists, it's value is base_path
store_path0=/opt/fdfs/fdfs_storage
#store_path1=/home/yuqing/fastdfs2
# store_path (disk), value can be 1 to 256, default value is 256
[geostar@host-173-16-122-223 fastdfs]#

删除上面base_path、store_path0路径的和文件,
注意,如果有未备份的文件，请先备份再删除
[geostar@host-173-16-122-223 fastdfs]# rm -rf /opt/fdfs/fdfs_storage

```
