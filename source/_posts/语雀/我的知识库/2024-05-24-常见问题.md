## 问题1、交互界面的黑窗口按<——Backspace删除键时出现了^H^H^H^H^H^H^H^H^H^H^H^H
```
解决办法：
^H不是H键的意思，是backspace
主要是当你的终端backspace有问题的时候才需要设置

在linux环境下使用sqlplus，在回删（backspace）时往往会出现 一串的乱码。
<!--more-->

解决方法有两种：

1。要使用回删键（backspace）时，同时按住ctrl键

2。设定环境变量

在bash下：$ stty erase ^?
或者把 stty erase ^? 添加到~/.bash_profile中。

在csh下：$ stty erase ^H
或者把 stty erase ^H  添加到~/.cshrc中。
```
## 问题二、zsh 和 bash 切换
### 从 bash 切换到 zsh
```bash
chsh -s /bin/zsh
```
### 从 zsh 切换回 bash
```bash
chsh -s /bin/bash
```
