1、使用快捷键"ctrl+H"或者找到“搜索”-“替换”，打开替换对话框
2、在查找目标栏输入：《.*DROP TABLE IF EXISTS.*\r\n》 在查找模式中选择”正则表达式“（如果需要删除包含DROP TABLE IF EXISTS的行，示例）
```
.*DROP TABLE IF EXISTS.*\r\n
```
