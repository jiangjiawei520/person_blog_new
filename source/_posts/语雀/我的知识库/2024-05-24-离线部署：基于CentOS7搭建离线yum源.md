##  一、目的
当集群部署在离线环境下，而系统又是比较纯净的时候，是没有yum源去安装相应依赖的。所以导致，安装程序很多失败。所以，决定自己创建私有的yum源，从而可以在离线环境中使用yum命令安装必备的文件。

## 二、步骤
1.下载包含yum源的镜像文件（即centos7.4安装文件）
进入下载初始页，选择对应版本7.4.1708
[http://vault.centos.org/](http://vault.centos.org/)
<!--more-->
选择Europe的第一个
[http://vault.centos.org/notonvault.html](http://vault.centos.org/notonvault.html)
选择对应版本，在此选择下载7.4
[http://mirror.nsc.liu.se/centos-store/7.4.1708/isos/x86_64/](http://mirror.nsc.liu.se/centos-store/7.4.1708/isos/x86_64/)
下载依据需求而定，最好下载everyting版本，但是因为太大，本文下载CentOS-7-x86_64-DVD-1708.iso版本
CentOS-7-x86_64-DVD-1708.iso
![](https://cdn.nlark.com/yuque/0/2021/png/12484160/1628842792346-c88c993b-1e2a-4d59-a756-1abe67e0884b.png#averageHue=%23f7f4f3&height=552&id=cfihj&originHeight=552&originWidth=1103&originalType=binary&ratio=1&rotation=0&showTitle=false&size=0&status=done&style=none&title=&width=1103)

2.上传到服务器
> /opt/softwares/CentOS-7-x86_64-DVD-1708.iso

3.挂着镜像到服务器
> mkdir /mnt/cdrom/
> mount /opt/softwares/CentOS-7-x86_64-DVD-1708.iso /mnt/cdrom/

4.将挂在盘内容复制到本地目录(非必须)
> mkdir /media/CentOS/

> cp -fr /mnt/cdrom/*  /media/CentOS/

5.将Yum源仓库repo备份
> mkdir /etc/yum.repos.d/bak
> mv /etc/yum.repos.d/* /etc/yum.repos.d/bak

备注：一定移除或者重命名/etc/yum.repos.d/CentOS-Base.repo这个文件，因为默认会首先读取这个访问外网源的文件

6.复制CentOS-Media.repo并且编辑
> cp /etc/yum.repos.d/bak/CentOS-Media.repo /etc/yum.repos.d/
> vi /etc/yum.repos.d/CentOS-Media.repo

修改为：

> # CentOS-Media.repo
> #
> #  This repo can be used with mounted DVD media, verify the mount point for
> #  CentOS-7.  You can use this repo and yum to install items directly off the
> #  DVD ISO that we release.

> #

> # To use this repo, put in your DVD and use it with the other repos too:

> #  yum --enablerepo=c7-media [command]

> #

> # or for ONLY the media repo, do this:

> #

> #  yum --disablerepo=\* --enablerepo=c7-media [command]

> [c7-media]

> name=CentOS-$releasever - Media

> baseurl=file:///media/CentOS/      #（1）-->这一行改成前边我们新建的CentOS目录

> #        file:///media/cdrom/      #（2）-->这一行加#号注释掉

> #        file:///media/cdrecorder/ #（3）-->这一行加#号注释掉

> gpgcheck=1

> enabled=1                          #（4）-->这一行把0改成1来启用源

> gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-CentOS-7

7.卸载镜像
> umount /mnt/cdrom/

8.检测
> yum list

## 三、共享yum源
1.在本地yum源配置完成之后安装apache
> yum install httpd

> systemctl start httpd

> systemctl enable httpd

> systemctl stop firewalld

> systemctl disable firewalld

2.创建挂载文件目录
> mkdir /var/www/html/source7.4

3.挂载镜像
> mount /opt/softwares/CentOS-7-x86_64-DVD-1708.iso  /var/www/html/source7.4/

4.开机自动挂载
> （1）编辑rc.local写入

> vim /etc/rc.d/rc.local

> mount /opt/softwares/CentOS-7-x86_64-DVD-1708.iso  /var/www/html/source7.4/

> （2）修改权限

> chmod +x /etc/rc.d/rc.local

5.访问
> http://IP/source7.4/
