## 0、备份nginx（很重要，有反悔机会）
```
cp -r nginx nginx20220801
```
## 1、nginx下载
到nginx官网下载最新稳定版本的nginx安装包。地址：[nginx: download](http://nginx.org/en/download.html)

## 2、下载
本地下载上传到服务器

## 3、解压
```
tar -zxvf nginx-1.22.0.tar.gz
```
## 4、编译
```
cd /usr/local/nginx-1.22.0
./configure --prefix=/usr/local/nginx
#如果是https的
#./configure --prefix=/usr/local/nginx --with-http_ssl_module


# 不要make install
make
```
## 5、备份历史版本、设置新版本、检查新版本
```
mv /usr/local/nginx/sbin/nginx /usr/local/nginx/sbin/nginx.old

cp /usr/local/nginx-1.22.0/objs/nginx /usr/local/nginx/sbin

cd /usr/local/nginx/sbin
./nginx -t
```
## 6、关闭历史版本nginx 
```
cd /usr/local/nginx/sbin
./nginx.old -s stop
```
## 7、启动新版本
```
./nginx  
```
## 8、检查nginx运行
```
ps -ef|grep nginx
```

1、如果历史nginx版本有安装过模块，请用./nginx -V 检查，并在 ./configure中追加上对应参数。
2、nginx版本升级不会覆盖配置文件；以防万一升级前请先备份配置文件。
