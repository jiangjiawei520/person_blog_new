## 一、Kettle7.1添加资源库（使用ORACLE作为资源库）
1.首先为资源库创建表空间
```plsql
create bigfile tablespace KETTLE_REPO datafile 'D:\app\Administrator\oradata\orcl\kettle_repo.dbf' SIZE 1024M autoextend on;
```
2建用户，授权限
```plsql
<!--more-->
--建用户，授权
create user kettle_repo identified by 123456 default tablespace KETTLE_REPO temporary tablespace temp;

grant connect,resource,dba to kettle_repo;
```

## 二、客户端打开后没有出现资源库的连接按钮解决办法
![image.png](https://cdn.nlark.com/yuque/0/2021/png/12484160/1630635288321-b4ad5eb4-f391-462f-a99f-734e0a67768c.png#clientId=u3e1cac69-c706-4&from=paste&id=uf637e31d&originHeight=871&originWidth=1424&originalType=url&ratio=1&size=100006&status=done&style=none&taskId=ub6a10fa8-e032-4009-96ca-1e2639392b7)
解决办法：

方法一：删除C:\Users\jiangjiawei\.kettle中repositories.xml文件，重启客户端：历史的资源库连接信息也丢失，需要重新填写
方法二： 将C:\Users\用户\.kettle中repositories.xml文件中的中文部分修改为英文，重启客户端。

注意：在创建资源库时请勿使用中文描述。
也可能是客户端JDK版本不是1.8导致的。
