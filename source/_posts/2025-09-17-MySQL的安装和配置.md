## 数据库的安装

1.打开下载的mysql安装文件双击解压缩，运行“mysql-5.5.40-win32.msi”。 

![img](/imgs/12487c40a0aef8744f608d224bb5afba.jpeg)

![img](/imgs/461adc624c0e19c77679f99cdb85ef5a.jpeg)

2.选择安装类型，有“Typical（默认）”、“Complete（完全）”、“Custom（用户自定义）”三个选项，选择“Custom”，按“next”键继续。 

![img](/imgs/77720f564128d211bfd10ead1b09e04e.jpeg)

<!--more-->

3.点选“Browse”，手动指定安装目录。

![img](/imgs/942b395a84b42952af8904051c0dbb97.jpeg)

 4.填上安装目录，我的是“d:\Program Files (x86)\MySQL\My[SQL Server](https://cloud.tencent.com/product/sqlserver?from_column=20065&from=20065) 5.0”，按“OK”继续。 

![img](/imgs/037bf392c67a6fd42892fa4b58a5f595.jpeg)

 5.确认一下先前的设置，如果有误，按“Back”返回重做。按“Install”开始安装。 

![img](/imgs/c79f50316be916ed415a1841067dbeed.jpeg)

![img](/imgs/d62d1bf1e4ab25b8b7c70acf258ffadd.jpeg)

![img](/imgs/091a139f410b6d8f40ebcc6c4199197d.jpeg)

![img](/imgs/cf1b3a3e77b99023462a3224618c6404.jpeg)

![img](/imgs/bc1c0cbff850945208308cf2c6b79d14.jpeg)

 6.正在安装中，请稍候，直到出现下面的界面, 则完成MYSQL的安装 

![img](/imgs/d804739c5e441e28918fcd1377b01fce.jpeg)

## 数据库的配置

1.安装完成了，出现如下界面将进入mysql配置向导。 

![img](/imgs/3ba478c46eabb43cda53fec245fe9f44.jpeg)

 2.选择配置方式，“Detailed Configuration（手动精确配置）”、“Standard Configuration（标准配置）”，我们选择“Detailed Configuration”，方便熟悉配置过程。 

![img](/imgs/9732c1e7658c4e6cbbc58724b9f58f20.jpeg)

 3.选择服务器类型，“Developer Machine（开发测试类，mysql占用很少资源）”、“Server Machine（服务器类型，mysql占用较多资源）”、“Dedicated MySQL Server Machine（专门的[数据库服务](https://cloud.tencent.com/product/tencentdb-catalog?from_column=20065&from=20065)器，mysql占用所有可用资源）” 

![img](/imgs/5bb6e2ec8c7b27faec8c733f81a6ea70.jpeg)

 4.选择mysql[数据库](https://cloud.tencent.com/product/tencentdb-catalog?from_column=20065&from=20065)的大致用途，“Multifunctional Database（通用多功能型，好）”、“Transactional Database Only（服务器类型，专注于事务处理，一般）”、“Non-Transactional Database Only（非事务处理型，较简单，主要做一些监控、记数用，对MyISAM数据类型的支持仅限于non-transactional），按“Next”继续。 

![img](/imgs/a30575272f04eca85e4b4be16181df1a.jpeg)

![img](/imgs/1c439c96b8c0e52013862583876a7405.jpeg)

 5.选择网站并发连接数，同时连接的数目，“Decision Support(DSS)/OLAP（20个左右）”、“Online Transaction Processing(OLTP)（500个左右）”、“Manual Setting（手动设置，自己输一个数）” 

![img](/imgs/d3f76570ff21ad8eaa889a5caa684a97.jpeg)

 6.是否启用TCP/IP连接，设定端口，如果不启用，就只能在自己的机器上访问mysql数据库了，在这个页面上，您还可以选择“启用标准模式”（Enable Strict Mode），这样MySQL就不会允许细小的语法错误。如果是新手，建议您取消标准模式以减少麻烦。但熟悉MySQL以后，尽量使用标准模式，因为它可以降低有害数据进入数据库的可能性。按“Next”继续 

![img](/imgs/46417c6ef27eab362c69a4bc21948a03.jpeg)

 7.就是对mysql默认数据库语言编码进行设置（重要），一般选UTF-8，按 “Next”继续 

![img](/imgs/1c17c822341fffee36065990eeb9fec1.jpeg)

 8.选择是否将mysql安装为windows服务，还可以指定Service Name（服务标识名称），是否将mysql的bin目录加入到Windows PATH（加入后，就可以直接使用bin下的文件，而不用指出目录名，比如连接，“mysql.exe -uusername -ppassword;”就可以了，不用指出mysql.exe的完整地址，很方便），我这里全部打上了勾，Service Name不变。按“Next”继续。 

![img](/imgs/a38e77d809df47a1d6f607791955ee78.jpeg)

 9.询问是否要修改默认root用户（超级管理）的密码。“Enable root access from remote machines（是否允许root用户在其它的机器上登陆，如果要安全，就不要勾上，如果要方便，就勾上它）”。最后“Create An Anonymous Account（新建一个匿名用户，匿名用户可以连接数据库，不能操作数据，包括查询）”，一般就不用勾了，设置完毕，按“Next”继续。 

![img](/imgs/1be1b1980d405d513ee6ac568472c5cd.jpeg)

 10.确认设置无误，按“Execute”使设置生效，即完成MYSQL的安装和配置。 

![img](/imgs/c4c29cb517d51ed3d1826ab4e11f3e60.jpeg)

![img](/imgs/986b59e98491d2e87eacaa2b5ac21ef2.jpeg)
